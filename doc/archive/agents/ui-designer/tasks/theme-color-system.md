# 设计任务：Win95 主题配色方案

**Agent**: UI Designer  
**创建日期**: 2026-03-17  
**状态**: 🟢 已完成  

---

## 设计需求

为 Cabin 项目确定一套完整的 Win95 风格配色方案，包括：
- 桌面背景色
- 窗口颜色系统
- 按钮状态颜色
- 语义化颜色（成功、警告、错误）

---

## 灵感参考

参考真实 Windows 95 界面：
- 默认桌面背景：`#008080` (Teal)
- 窗口背景：`#c0c0c0` (Silver)
- 文本颜色：`#000000` (Black)
- 选中高亮：`#000080` (Navy)

---

## 配色方案

### 主色调

| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 桌面背景 | `#008080` | 经典 Teal 色 |
| 窗口背景 | `#c0c0c0` | Silver 灰色 |
| 内容区域 | `#ffffff` | 纯白背景 |
| 主要文本 | `#000000` | 纯黑文本 |

### 按钮颜色系统

| 状态 | 颜色值 | 边框效果 |
|------|--------|----------|
| 正常态 | `#c0c0c0` | 上左白，下右灰 |
| Hover 态 | `#d0d0d0` | 轻微变亮 |
| 按下态 | `#a0a0a0` | 反转边框 |
| 禁用态 | `#e0e0e0` | 降低对比度 |

### 语义化颜色

| 类型 | 颜色值 | 使用场景 |
|------|--------|----------|
| 成功 | `#00ff00` | 操作成功提示 |
| 警告 | `#ffff00` | 警告信息 |
| 错误 | `#ff0000` | 错误提示 |
| 信息 | `#0000ff` | 链接和提示 |

| 组合 | 对比度 | 标准 | 结果 |

## 样式代码

```javascript
// React95 主题扩展
const customTheme = {
  ...themes.default,
  background: '#008080', // Teal 桌面
  
  window: {
    face: '#c0c0c0',
    shadow: '#808080',
    highlight: '#ffffff',
  },
  
  button: {
    face: '#c0c0c0',
    shadow: '#808080',
    highlight: '#ffffff',
  },
};
```

## 开发对接说明

1. 使用 React95 的 `ThemeProvider` 应用主题
2. 桌面背景色在 Desktop 组件中设置
3. 所有颜色使用 CSS 变量便于维护

# Win95 风格个人网站开发总结

## 项目背景

这个项目旨在创建一个怀旧的 Windows 95 风格个人网站，
让用户能够在一个模拟的桌面环境中浏览博客内容。

## 技术选型

- **Next.js**: React 框架，支持静态导出
- **React95**: Win95 风格 UI 组件库
- **Styled-components**: CSS-in-JS 样式方案

## 核心功能

1. **桌面环境**
   - 任务栏
   - 开始菜单
   - 桌面图标

2. **窗口系统**
   - 拖拽移动
   - 最小化/最大化
   - 窗口聚焦

3. **文件浏览器**
   - 文件夹导航
   - 文件图标显示

4. **记事本**
   - Markdown 支持
   - 滚动查看

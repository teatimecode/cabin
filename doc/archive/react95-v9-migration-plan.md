## 已知问题

### 高优先级

1. **博客文章内容未加载**
   - 问题：双击博客文章文件无法显示内容
   - 原因：FileExplorer 的 onOpenItem 回调未正确传递内容
   - 解决方案：需要在 FileManager 中读取 postId 并加载内容

2. **窗口大小调整缺失**
   - 问题：窗口无法通过拖拽边框调整大小
   - 影响：用户体验不完整
   - 解决方案：添加边框拖拽处理逻辑

### 中优先级

3. **右键菜单未实现**
   - 问题：右键点击无响应
   - 影响：缺少上下文菜单功能
   - 解决方案：实现通用的右键菜单组件

4. **快捷键支持不完整**
   - 问题：只能通过按钮操作窗口
   - 影响：键盘用户效率低
   - 解决方案：添加全局快捷键监听
